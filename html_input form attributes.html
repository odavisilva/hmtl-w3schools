O atributo de novalidate do formulário

O atributo novalidate especifica que os dados do formulário não devem ser validados ao serem enviados.



Observação: o atributo novalidate da tag do formulário não é compatível com Safari 10 (ou anterior).

O atributo de entrada do formnovalidate



Note: The formnovalidate attribute of the input tag is not supported in Safari 10 (or earlier).

O atributo input formtarget

O atributo formtarget especifica um nome ou uma palavra-chave que indica onde exibir a resposta recebida após o envio do formulário.





O atributo do método de entrada do formulário

O atributo formmethod define o método HTTP para enviar dados do formulário ao URL de ação.





O atributo input formenctype

O atributo formenctype especifica como os dados do formulário devem ser codificados quando enviados.



O atributo de formação de entrada

O atributo formaction especifica o URL de um arquivo que processará a entrada quando o formulário for enviado.





O atributo do formulário de entrada

O atributo form especifica o formulário ao qual um elemento de entrada pertence.



O campo "Sobrenome" abaixo está fora do elemento do formulário, mas ainda faz parte do formulário.
