O elemento da imagem

Redimensione o navegador para ver diferentes versões da imagem sendo carregada em diferentes tamanhos de janela de visualização. O navegador procura o primeiro elemento de origem onde a consulta de mídia corresponde à largura da janela de visualização atual do usuário, e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filha do bloco de declaração da imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento de imagem, ou se nenhuma das tags de origem corresponder.

Observação: o elemento de imagem não é compatível com IE12 e anterior ou Safari 9.0 e anterior.

O elemento da imagem

O elemento de imagem pode ser usado quando o formato da imagem não é compatível com todos os dispositivos.

O dispositivo usará o primeiro formato de imagem compatível e ignorará o restante das imagens.
